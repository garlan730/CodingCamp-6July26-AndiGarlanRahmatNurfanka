A responsive **Expense & Budget Visualizer** built with **HTML, CSS, and Vanilla JavaScript** that helps users track their daily spending, monitor their total balance, and visualize expenses by category through an interactive pie chart.

This project was developed as the final project for the **5-Day Frontend Web Development Bootcamp by RevoU**. The challenge was to build a complete, client-side expense tracking application using only fundamental frontend technologies without any frameworks or backend services.

During the development process, I also leveraged **Kiro AI** as an AI-assisted development tool for brainstorming ideas, learning best practices, debugging issues, and improving the application's user experience. All features, implementation decisions, and customizations were built and understood as part of my learning journey throughout the bootcamp.

---

## 📊 How It Works

### Adding a Transaction

1. Enter an item name.
2. Enter an amount.
3. Select a category.
4. Click **Add Transaction**.

The application will automatically:

- Validate the input
- Save the transaction
- Update the balance
- Update the transaction history
- Update the pie chart
- Persist data in Local Storage

---

### Deleting a Transaction

Users can remove any transaction by clicking the **Delete** button. The application automatically updates:

- Total balance
- Transaction list
- Pie chart
- Local Storage

---

### Sorting Transactions

Transactions can be sorted by:

- Latest
- Amount
- Category

---

### Dark Mode

The selected theme is automatically remembered even after refreshing the page.
